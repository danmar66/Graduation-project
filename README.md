```
cd frontend
npm i

```

## Start backend server

Make sure you are in the **`backend`** folder. Then run server.

```
npm run dev
```

## Start frontend client

Go to project **`root`** directory. Then open **`frontend`** folder. After this run frontend client.

```
cd ../frontend/
```
